函数声明与表达式

在前面学过函数主要的两种命名方式

函数声明

  • 通过这样的方式声明，会得到函数声明提升，即可以在声明之前使用该函数
  • 另外在 Firefox，Safari，Chrome，Opera 中提供了一个 name 属性，用于获取函数名字。仅是这四个浏览器拥有

由于存在函数声明提升，所以不要试图用条件语句去多次声明函数，因为无论什么条件下，函数声明方式声明的多个同名函数，只会解析最后命名的那一个

不同浏览器处理方式不一，建议不要这么写

函数表达式

  • 通过将一个未命名的函数声明赋值给一个变量来完成。不存在函数声明提升，必需要在函数表达式之后调用
  • 由于该方式使用的方式无需声明函数名，只需赋值给变量，所以又叫 匿名函数
