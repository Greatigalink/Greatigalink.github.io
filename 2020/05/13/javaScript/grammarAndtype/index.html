语句

  • Js的语句结尾分号不是必须的。
  • 但是需要注意，加上分号可以避免一些错误。

变量

数据类型

typeof操作符

它并非是一个函数而是操作符，所以括号可加可不加

Undefined 类型

常见的undefined就是属于这种类型，它也只有这一种值。通常申明变量但没有赋值就会默认为undefined

Null 类型

同样的它也只有一个特殊值null。同时null值表示一个空指针对象。

Boolean 类型

它拥有两个字面值true，false，true不一定为1,false也不一定为0。

注意True和False是标识符，因为这里的true，false也是区分大小写的。

Number 类型

1、基本初始化

十进制整数

八进制

  • 首位必为0
  • 八进制序列为0 ~ 7，所以有数值一旦不属于这个范围，前导首位的0会被忽略，整个数会被当做十进制处理
  • 严格模式下八进制字面量无效

十六进制

  • 前两位必为0x
  • 后面可以为（0 - 9 及 A - F），字母A - F可小写

在进行算术计算时，八进制与十六进制都会转为十进制数进行计算

2、浮点数

定义

  • 小数点后至少有一位数字
  • 小数点前可以没有整数，但建议不要省略

科学计数

对于小数点后有6个零以上的，会转为科学计数法

两个小细节

（1）对于小数点后没有数字或都为零时，ES会将其当成整数看待，因为浮点数所需内存为整数的两倍

（2）最好不要做类似 ‘x + y == 0.3’ 这样的比较！来看个例子

  • 你会发现第一条和第三条不是意想中的结果，if语句里也不执行呢？why？
  • 因为浮点值最高精度为17位小数，计算的过程中会导致小小的舍入误差，所以有时准确，有时就差那么一点

3、数值范围

  • 最小值存在 “Number.MIN_VALUE” 中,值为 5e-324
  • 最大值存在 “Number.MAX_VALUE” 中,值为 1.7976931348623157e+308
  • 超出这个范围，数据会被转为 Infinity，负的就为 -Infinity
  • isFinite()函数可以测试数据是否在范围内，在就返回true，反之false

4、NaN

  • NaN是一个特殊值，本来是要返回数值却没有返回，就用它来替代
  • NaN参与的任何操作都会返回NaN
  • NaN不等于包括他在内的所有值
  • isNaN()函数用来测试一个数是否”不是数值”或者是否能转为数值

5、数值转换

三个函数用于将非数值转为数值

（1）Number()

此函数用于任何数据类型

  • true, false转为1, 0
  • null，转0
  • undefined，转NaN
  • 字符串只含数字的，转为相应数值

（2）parseInt()

  • 如果第一个字符为非数值，直接返回NaN，否则从第一个数值开始解析，直到碰见第一个非数符就停止
  • 如果是八进制或者十六进制的字量

由于ES3和ES5存在分歧，所以对于数解析有差异

所以parseInt()函数还提供了一个参数，用于指定转换为何种进制

（3）parseFloat()

  • 始终忽略前导零，整个串中只解析第一个小数点，若遇到第二个小数点就停止解析
  • 只解析十进制数，所以不像parseInt()有第二个参数
  • 若解析的串包含一个可解析为整数的串就会返回为整数

String 类型

初始化单引号或双引号都行

1、字符字面量

String数据类型包含一些特殊的字符面量，其实就是常称的转义符或转义序列。

  • 比如 \n \t \b 等等

任何字符串都可以通过它自己的length属性得到长度

2、字符串拼接

  • 创建一个能容纳两个待拼接字串大小的新字符串
  • 将拼接后的字串放进这个新串
  • 销毁原来的两个字符串

3、字符串的转换

toString( )方法

  • 几乎所有的值都可以用此方法转为字符串
  • 对于数值，该方法提供一个参数以指定转为某种进制的展现方式
  • null 和 undefined 没有该方法

String( )函数

  • 可以将任何值转为字符串
  • 如果值有toString()方法，则实际上调用的是该方法
  • 所以此函数其实是为了null 和 undefined 服务
