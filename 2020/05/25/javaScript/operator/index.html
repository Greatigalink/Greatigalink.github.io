操作符

一元操作符

1、递增操作符

  • ++，-- 与大部分语言类似，分为前置和后置
  • 即 --a, ++a 或者 a--, a++

注意

  • 对含有效数字字符串和布尔值true,false，会将转为数值在进行递增操作
  • 应用于浮点数执行加减1操作
  • 对于对象，利用valueof( )方法获取有效数值在进行操作
  • 若都不满足上述，一般会用NaN代替原来数值
  • 这个跟parseInt()和parseFloat()不一样，字串但凡包括非数字直接为NaN,不像这两个方法开头为数字也可解析

2、一元加减

  • 正常加减无异
  • 在字符串前加上+或-,会将其转为数值，若转不了则为NaN
  • 这个跟parseInt()和parseFloat()不一样，字串但凡包括非数字直接为NaN,不像这两个方法开头为数字也可解析

位操作符

计算机反码、补码不在陈述

按位非、与、或、异或

  • 这些操作在底层是把数值转为二进制后进行的
  • 位操作符会把NaN和Infinity当做数值0处理
  • 非：~，求反码
  • 与：&，两个数的与操作
  • 或：|，两个数的或操作
  • 异或：^，两个数的异或操作

布尔操作符

1、逻辑非(!)

  • 操作数为对象、非空字符串、任意非零数值（包括Infinity）返回false
  • 操作数为空字符串、数值0、null、NaN、undefined返回true

2、逻辑与(&&)

  • 第一个操作数为对象、返回第二个操作数，反之只有第一个操作数为true时才返回该对象
  • 两个都为对象返回第二个
  • 第一个操作数为null、NaN、undefined，返回它本身

3、逻辑或(||)

  • 第一个操作数为对象、返回第一个操作数，反之只有第一个操作数为false时才返回该对象
  • 两个都为对象返回第一个
  • 第一个操作数为null、NaN、undefined，返回它本身
