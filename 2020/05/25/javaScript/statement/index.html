语句

又称流控制语句，这里只写 for-in 、 label 、with 语句

for-in

精准迭代语句，枚举对象属性

  • 可以用于遍历数组的元素–>使用for-in语句遍历数组元素的时候遍历出来是数组的下标
  • 可以用于遍历对象的所有属性数据–>使用for-in语句遍历对象的属性的时候，遍历出来的是对象的属性名

label

  • 类似C语言中的 go to 语句

with

with语句的作用是将代码的作用域设置到一个特定的对象中。目的主要是为了简化多次编写同一个对象的工作。
具体结合后续章节作用域链来理解

函数 function

函数定义

  • return 语句后的代码不会再执行,若 return 不返回任何定义的值，则返回 undefined

参数

  • 函数创建时可以不用声明参数名
  • 传入的参数被放在arguments对象里，可在函数体内访问，若声明了参数名也可用参数访问
  • arguments 对象与传入的参数值是一一对应的，比如 arguments[0] 对应第一个参数……以此类推
  • 当改变 arguments 的值时，对应的参数值也会改变，反之参数值变了 arguments 也会变

注意，虽然数据看似像是操作同一个变量，事实上 arguments 和参数是相互独立的
