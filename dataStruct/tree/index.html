树

树形结构是一种很重要的数据结构，它有几个基本属性

  • 根节点：整个树的初始节点
  • 子节点：某个节点下拥有的 n 个节点
  • 父节点：某个或某些节点共有的上一层节点
  • 兄弟节点：它们拥有同一个父节点
  • 叶子节点：该节点下没有其他节点
  • 层：从根节点开始，根节点为第一层，根的孩子为第二次依次类推
  • 深度：树的层数
