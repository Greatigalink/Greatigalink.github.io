网络基础 2020-07-15|计算机网络|URI与URL-TCP/IP

分层管理

从上到下

  • 应用层：负责处理像 http 这样的报文，直接对口应用程序
  • 传输层：负责处理 TCP 这样的可靠连接，通过三报文，端口的方式确保传输的报文完整，对口上层应用层
  • 网络层：也就是 IP 层，通过发送一定大小的数据报来传送传输层给到的数据，不保证可靠，但会尽力发送更多的报文
  • 链路层：负责管理 IP 地址与各种网络实体设备的映射，确保找到正确，最短的传输路径

基本包装类型与内置对象 2020-07-04|JavaScript|引用类型-Global-Math

基本包装类型

基本包装类型主要针对 String, Boolean, Number，他们虽然是基本类型值，但是也有创建他们的实例方式，

  • 一般情况下，采用字面量的方式定义时，他们就是基本类型值
  • 可以采用 new Object() 或者对应的构造方法创建，此时就是对象

我们知道它们拥有各自的一些方法，但是基本类型值本身只是一个值并没有方法，所以在调用基本类型值的方法时，后台会创建各自的实例并传入他们的值，在调用实例的方法之后又消除实例，回到原本的基本类型值。
