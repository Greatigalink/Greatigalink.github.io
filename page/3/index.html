浏览器BOM核心 2020-07-20|浏览器BOM|window对象

window 对象

全局作用域

这个不用多讲，前面已经接触过，所有在全局作用域定义的变量都会被当做 window 对象的属性，同时 Global 对象也是基于 window 对象的。

需要注意的是，一般定义变量时，这个变量的四个数据属性都默认为 true（这个前面面向对象那里讲过），所以都是可以删除或者修改的。**可是不一样的地方就在全局作用域，在此作用域下定义的变量的数据属性之一 [[Configurable]]**默认为 false，这就导致无法删除，强行使用 delete 是无效的，在 IE9 之前的浏览器中还会报错

网络基础 2020-07-15|计算机网络|URI与URL-TCP/IP

分层管理

从上到下

  • 应用层：负责处理像 http 这样的报文，直接对口应用程序
  • 传输层：负责处理 TCP 这样的可靠连接，通过三报文，端口的方式确保传输的报文完整，对口上层应用层
  • 网络层：也就是 IP 层，通过发送一定大小的数据报来传送传输层给到的数据，不保证可靠，但会尽力发送更多的报文
  • 链路层：负责管理 IP 地址与各种网络实体设备的映射，确保找到正确，最短的传输路径
